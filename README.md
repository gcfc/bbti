# BBTI — Basketball Behavioral Type Indicator

A satirical MBTI-style personality test for basketball players. 30 questions, 6 dimensions, 20 personality types.

## Features

- **Bilingual**: English / Chinese toggle with full translations
- **30 humorous questions** across 6 basketball dimensions
- **20 personality types** from KOBE (The Mamba) to BENCH (The Professional Benchwarmer)
- **Zero dependencies** — vanilla HTML, CSS, and JavaScript
- **Mobile responsive**

## Dimensions

| Dim | Name | Low | High |
|-----|------|-----|------|
| S | Shot Selection | Chaotic Chucker | Calculated Assassin |
| D | Defense | Traffic Cone | Lockdown Warden |
| E | Ego | Role Player Energy | Main Character Syndrome |
| H | Hustle | Load Management | Leave It All On The Court |
| I | Court IQ | Vibes Only | Chess Master |
| C | Clutch | Brick Under Pressure | Ice In Veins |

## Deployment

Static site — serve the `bbti/` folder from any static host. No build step required.

For GitHub Pages, point the deployment source to this folder.
